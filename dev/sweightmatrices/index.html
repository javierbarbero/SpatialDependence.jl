Spatial Weight Matrices

Polygon contiguity

SpatialDependence.polyneigh — Function
polyneigh(P, criterion = :Queen)

Build a spatial weights object from a vector of polygons P.

Optional Arguments

  • criterion=:Queen: neighbour criterion. :Queen or :Rook.
  • tol=0.0: tolerance for polygon contiguity.

Examples

julia> using SpatialDatasets

julia> guerry = sdataset("Guerry");

julia> W = polyneigh(guerry.geometry)
Spatial Weights object 
Observations: 85 
Transformation: binary
Average number of neighbors: 4.9412
Minimum nunmber of neighbors: 2
Maximum nunmber of neighbors: 8
Median number of neighbors: 5.0
Islands (isloated): 0
Density: 5.8131%

Spatial Matrix transformation

SpatialDependence.wtransform! — Function
wtransform!(W::SpatialWeights, style::Symbol)

In-place transformation of the weights using the specified style.

Weights transformation

style can be one of the following:

  • :binary: 1 if neighbor, 0 if not.
  • :row: row-standardized. Each row sum equals one.

Examples

julia> W = SpatialWeights([0 1 0; 1 0 1; 0 1 0]);

julia> wtransform!(W, :row)
source
